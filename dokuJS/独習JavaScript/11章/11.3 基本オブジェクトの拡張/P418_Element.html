Element

Elementオブジェクトの生成

var a = new Element('a', 
  { 'class': 'link', href: 'http://www.google.co.jp/'
}).update("Google");

CSSのスタイルクラスの操作

// クラス名を追加
$('contents').addClassName('class3');

// クラス名を削除
$('contents').removeClassName('class1');

Prototype.js にはクラス名の追加や削除のための便利なメソッドが用意されており、 直観的に操作ができます。
